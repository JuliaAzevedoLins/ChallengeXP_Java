Investaê

Bem-vindo(a) à Investaê, a API RESTful para gerenciamento de usuários investidores, investimentos, rentabilidades diárias, bancos e tipos de investimento.

Esta aplicação foi desenvolvida em Java 17 com Spring Boot 3, seguindo as melhores práticas de arquitetura, documentação e segurança.

Documentação Swagger Repositório no GitHub

Como usar?

Utilize a documentação Swagger para explorar e testar todos os endpoints disponíveis.
Para dúvidas, consulte o README no repositório.
